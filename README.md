# 🚗 车辆技术深度学习平台

基于18年汽车制造经验构建的系统化学习平台，涵盖新能源汽车技术、制造工艺与质量管理体系。

## 🌐 在线访问
**主页面**: https://maxiaopeng98.github.io/vehicle-tech-learning/

## 📚 学习内容

### 技术路线深度学习
1. **🔋 纯电动汽车 (BEV)** - 三电系统、制造工艺、质量管理
2. **⚡ 插电混动汽车 (PHEV)** - 双动力系统、工作模式、技术特点
3. **🔄 增程式电动汽车 (EREV)** - 增程器原理、能量管理、NVH控制
4. **⛽ 传统燃油车 (ICE)** - 内燃机技术、制造工艺、转型路径
5. **🚀 燃料电池汽车 (FCEV)** - 氢能技术、燃料电池、安全系统

### 质量管理体系
6. **✅ 质量管理体系** - ISO9001、IATF16949、五大工具、过程审核

## 🎯 学习特色

### 系统化学习框架
每个技术路线包含9个标准模块：
1. **知识概述** - 基本概念、学习目标
2. **定义与分类** - 准确定义、技术分类
3. **工作原理** - 系统架构、能量流动
4. **核心组件** - 关键部件、技术参数
5. **材料科学** - 材料选择、性能要求
6. **制造工艺** - 生产流程、工艺难点
7. **质量管理** - 体系要求、控制要点
8. **市场应用** - 现状分析、发展趋势
9. **学习资源** - 参考资料、实践建议

### 结合制造经验
- 基于18年汽车制造实践经验
- 重点关注制造工艺与质量控制

## 🚀 使用方式

### 在线学习
1. 访问 https://maxiaopeng98.github.io/vehicle-tech-learning/
2. 选择感兴趣的技术路线
3. 点击"深入学习"按钮进入详细页面
4. 使用侧边栏导航系统学习各个模块

### 本地学习
1. 克隆仓库到本地
2. 用浏览器打开 `index.html`
3. 所有功能均可离线使用
