2. 定义与分类

PHEV 的准确定义、技术分类及标准规范

🔍 技术定义

插电混动汽车 (Plug-in Hybrid Electric Vehicle, PHEV) 是一种兼具纯电动和燃油车特点的混合动力汽车：

  • 可外接充电：可通过充电桩为动力电池充电
  • 双动力系统：发动机和电机可独立或协同驱动车辆
  • 多种工作模式：纯电、混动、燃油模式智能切换
  • 长纯电续航：纯电续驶里程≥50km（国标要求）
  • 低油耗：综合油耗显著低于传统燃油车

国家标准定义 (GB/T 19596-2017)：具有可外接充电功能，且最大电功率比≥30% 的混合动力汽车。

3. 工作原理

PHEV 的系统架构、工作模式及控制策略

🔄 系统架构

PHEV 系统架构主要包括以下核心组件：

  • 发动机系统：内燃机，提供机械能或发电
  • 电机系统：驱动电机，将电能转换为机械能
  • 动力电池系统：存储电能，为电机供电
  • 电控系统：电机控制器、电池管理系统
  • 整车控制器 (VCU)：协调各系统工作
  • 变速箱：单档/多档变速机构

并联式 PHEV 能量流动示意图

⛽ 燃油 → 🔥 发动机 → 🛞 车轮
🔋 电池 → ⚡ 电机 → 🛞 车轮

智能控制策略：

  • SOC 平衡控制：保持电池电量在合理区间
  • 基于导航的预测控制：根据路线提前调整模式
  • 基于路况的自适应控制：根据拥堵情况优化策略
  • 基于成本的优化控制：根据电价/油价选择最优模式

4. 核心组件

PHEV 关键部件的技术参数、功能要求及选型原则

🔋 动力电池系统

技术要求：

  • 高功率密度：≥1500 W/kg，满足加速和回收需求
  • 长循环寿命：≥3000 次，适应频繁充放电
  • 高安全性：通过针刺、挤压、过充等测试
  • 宽温度适应性：-30℃~55℃正常工作
  • 快速充电：支持 3-6kW 交流充电

电池类型对比：

电池类型能量密度循环寿命成本安全性应用
三元锂 (NCM)高 (200-280)中 (1500-2000)高中高端 PHEV
磷酸铁锂 (LFP)中 (160-200)高 (3000+)低高主流 PHEV
刀片电池中高 (180-220)高 (3000+)中很高比亚迪 DM-i

⚡ 驱动电机系统

电机类型选择：

  • 永磁同步电机 (PMSM)：效率高、功率密度高，主流选择
  • 交流异步电机 (IM)：成本低、高速性能好
  • 开关磁阻电机 (SRM)：结构简单、成本低，但噪音大

PHEV 电机特点：

驱动电机

功率：100-200 kW

冷却方式：水冷

发电机

功率：50-100 kW

转速：3000-6000 rpm

效率：≥95%

控制方式：矢量控制

电控系统

IGBT/SiC 模块

开关频率：10-20 kHz

防护等级：IP67

冷却方式：水冷

🔥 发动机系统

PHEV 专用发动机特点：

  • 高热效率：≥43%，降低油耗
  • 阿特金森循环：提高热效率，牺牲部分功率
  • 取消皮带轮系：由电机直接驱动附件
  • 快速暖机：降低冷启动排放和油耗
  • 低摩擦设计：降低机械损失

典型 PHEV 发动机参数：

技术参数数值范围说明
排量1.0-2.0L小排量涡轮增压为主
最大功率80-150 kW与电机协同工作
最大扭矩150-300 N·m低转速高扭矩
热效率40-45%混动专用发动机
压缩比13-16:1高压缩比提高效率

🔌 混动专用变速箱

PHEV 变速箱类型：

  • 单档减速器：结构简单，成本低（比亚迪 DM-i）
  • 多档 DHT：2-3 档，兼顾动力和油耗（长城柠檬 DHT）
  • E-CVT：功率分流，平顺性好（丰田 THS）
  • 双离合 DCT：传动效率高，换挡快（大众 GTE）

5. 材料科学

PHEV 关键材料的性能要求、选择标准及发展趋势

🔋 电池材料

正极材料：

  • 磷酸铁锂 (LFP)：安全性好，循环寿命长，成本低
  • 三元材料 (NCM)：能量密度高，但热稳定性较差
  • 锰酸锂 (LMO)：成本低，安全性好，但循环寿命短

负极材料：

  • 石墨：技术成熟，成本低，容量 372 mAh/g
  • 硅碳复合：容量高（4200 mAh/g），但体积膨胀

6. 制造工艺

PHEV 关键部件的生产工艺、装配流程及质量控制要点

🔋 电池包制造工艺

电芯制造流程：

  1. 电极制备：混料→涂布→辊压→分切
  2. 装配：卷绕/叠片→入壳→注液→封口
  3. 化成：首次充电→老化→分容
  4. 检测：OCV、内阻、自放电测试

  • 叠压：自动叠铆，叠压系数≥0.97
  • 绕组：发卡绕组/传统绕组
  • 浸渍：VPI 真空压力浸渍

🔥 发动机与总装工艺

发动机装配线：

  1. 缸体装配：主轴瓦→曲轴→活塞连杆
  2. 缸盖装配：气门→凸轮轴→摇臂
  3. 总成装配：缸盖合装→正时→附件
  4. 测试：冷试→热试→排放测试

整车总装工艺：

  • 底盘合装：电池包、电机、发动机同步安装
  • 高压系统安装：橙色线束、安全标识
  • 燃油系统安装：油箱、油管、碳罐
  • EOL 测试：功能测试、路试、淋雨测试
